# Управление
---

* ###  Управление камерой
	*  <span style="color:#f55">Колёсико мыши (зажать)</span> --> Вращать [[camera#Поведение в point-and-click Point and Click|камеру]] по горизонтали
	*  <span style="color:#f55">Колёсико мыши (вращать)</span> --> Приблизить/отдалить [[camera#Поведение в point-and-click Point and Click|камеру]]
	*  <span style="color:#f55">Колёсико мыши (нажать)</span> --> Сменить фокус [[camera#Поведение в point-and-click Point and Click|камеры]] на объект под курсором
	
* ###  Ходьба ^ad42e9
	* *Шерон автоматически идет к каждой новой метке*
	*  ( <span style="color:#f55">Левая кнопка мыши</span> ) --> Поставить метку
	*  ( <span style="color:#f55">C</span> ) --> Удалить метку и остановить Шерон
	*  ( <span style="color:#f55">Z</span> ) --> Красться
		*  *Режим переключения состояния (переключатель/рация) можно изменить в настройках*
	
* ###  Инвентарь
	*  ( <span style="color:#f55">1</span> - <span style="color:#f55">9</span> ) --> Переключить активный слот инвентаря
	*  ( <span style="color:#f55">Ctrl</span> + <span style="color:#f55">Колесо мыши (вращать)</span> ) --> Быстрое переключение активного слота
	
* ###  Смена игрового режима
	*  ( <span style="color:#f55">V</span> ) --> Переключиться в режим [[beat-em-up|Beat Em Up]]
	
#  Способности
---

* ###  ( <span style="color:#f55;display:inline">Правая кнопка мыши</span> ) - Открыть контекстное меню взаимодействия с объектом под курсором
	*  Осмотреть - Шерон прокомментирует выбранный объект
	*  Взаимодействовать - виды взаимодействий варьируются:
		*  Приближенный осмотр - можно крутить объект в 3D/проиграется катсцена
		*  Головоломка - открывается [[menus#Интерфейс головоломки|интерфейс головоломки]]
		*  Взлом - камера переключается на [[menus#Интерфейс взлома|мини-игру взлома]]
		*  Вид - Шерон и все персонажи поблизости сядут спиной к игроку и будут смотреть вдаль, игровой интерфейс плавно затухает до первого ввода пользователя. Камера переходит в красивый ракурс. (толчок сделать снимок экрана)
	*  Диалог - запускает [[menus#Интерфейс диалога|интерфейс диалога]]
	*  Обыскать - спустя время применения, открывает [[menus#Интерфейс хранилища|интерфейс хранилища]]
	*  Подобрать - перемещает выбранный объект в инвентарь
	*  Применить - Шерон применит предмет в активном слоте на выбранном объекте
	*  Выбросить - выкинуть предмет из инвентаря
	*  Спрятаться - Шерон прячется в выбранное укрытие
	*  Пройти - сменить комнату (все персонажи пройдут следом)
	
* ###  Активные способности

		*  *Помечает все несекретные объекты в поле зрения, доступные для взаимодействия*

# Механики
---

* ###  Скрытность
	*  *Шерон не атакуют враги пока она не попадет им в поле зрения*
	*  *Шерон хуже видно в темноте*
	*  *Шерон хуже видно в маскировке*
	*  *Шерон легче заметить в режиме [[beat-em-up|Beat Em Up]]*
	*  *Шерон легче заметить при движении*
	*  *Использование громких/ярких предметов привлекает внимание, но не сразу выдает Шерон*
	*  *Нейтральные враги начинают битву, когда понимают что перед ними полицейский*
	*  *[[enemies#^77edde|Агрессивные враги]] атакуют Шерон сразу, как заметят её*
