# Управление
---

* ###  Управление камерой
	*  ( <span style="color:#f55">Зажать колесо мыши</span> ) --> Вращать камеру
	*  ( <span style="color:#f55">Вращать колесо мыши</span> ) --> Приблизиться/отдалить
	*  ( <span style="color:#f55">Нажать среднее колесо мыши</span> ) --> Сфокусироваться на объекте под курсором
	*  ( <span style="color:#f55">Зажать Ctrl</span> ) --> Выделить подконтрольных персонажей курсором
	
* ###  Ходьба
	* *Персонаж автоматически идет к его метке*
	*  ( <span style="color:#f55">Левая кнопка мыши</span> ) --> Поставить метку
	*  ( <span style="color:#f55">C</span> ) --> Удалить метку и остановить всех фокусируемых персонажей
	*  ( <span style="color:#f55">Зажать/нажать Shift*</span> ) --> Красться
		*  *Режим переключения состояния можно изменить в настройках*
	


#  Способности
---

* ###  Взаимодействие
	*  ( <span style="color:#f55">Правая кнопка мыши</span> ) - Открыть контекстное меню взаимодействия с объектом под курсором
		*  Осмотреть - контролируемый персонаж прокомментирует выбранный объект
		*  Взаимодействовать - виды взаимодействий варьируются:
			*  Приближенный осмотр - можно крутить объект в 3D/проиграется катсцена
			*  Головоломка - открывается [[ui#Интерфейс головоломки|интерфейс головоломки]]
			*  Взлом - камера переключается на [[ui#Интерфейс взлома|мини-игру взлома]]
			*  Вид - все подконтрольные персонажи сядут спиной к игроку и будут смотреть вдаль, игровой интерфейс плавно затухает до первого ввода пользователя. Камера переходит на вид. (толчок сделать снимок экрана)
		*  Диалог - запускает [[ui#Интерфейс диалога|диалог]]
			*  *Доступно только Шерон!*
		*  Обыскать - спустя время применения, открывает [[ui#Интерфейс хранилища|интерфейс хранилища]]
		*  Подобрать - перемещает выбранный объект в инвентарь
		*  Спрятаться - подконтрольный персонаж прячется в выбранное укрытие
		*  Пройти - сменить комнату (все подконтрольные персонажи пройдут следом)
	
* ###  Инвентарь
	*  ( <span style="color:#f55">Z</span> ) --> Открыть/закрыть [[ui#Интерфейс инвентаря|инвентарь]]
	
* ###  Активные способности
	*  ( <span style="color:#f55">Пробел</span> ) --> Осмотреться
		*  *Помечает все несекретные объекты в поле зрения, доступные для взаимодействия*
	*  ( <span style="color:#f55">X</span> ) --> Применить предмет в активном слоте на объекте бод курсором
