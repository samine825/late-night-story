# Управление
---

* ###  Управление камерой
	*  ( <span style="color:#f55">Зажать колесо мыши</span> ) --> Вращать камеру
	*  ( <span style="color:#f55">Вращать колесо мыши</span> ) --> Приблизиться/отдалить
	*  ( <span style="color:#f55">Нажать среднее колесо мыши</span> ) --> Сфокусироваться на объекте под курсором
	
* ###  Ходьба
	* *Шерон автоматически идет от метки к метке*
	*  ( <span style="color:#f55">Левая кнопка мыши</span> ) --> Поставить метку
	*  ( <span style="color:#f55">S</span> ) --> Удалить метку и остановить Шерон
	*  ( <span style="color:#f55">Зажать D</span> ) --> Приостановить Шерон
	*  ( <span style="color:#f55">Зажать/нажать Shift*</span> ) --> Бежать
		*  *Можно изменить в настройках*
	*  ( <span style="color:#f55">Зажать/нажать Ctrl*</span> ) --> Красться
		*  *Можно изменить в настройках*
