# Управление
---

* ###  Управление камерой
	*  ( <span style="color:#f55">Колёсико мыши (зажать)</span> ) --> Вращать камеру
	*  ( <span style="color:#f55">Колесико мыши (вращать)</span> ) --> Приблизиться/отдалить
	*  ( <span style="color:#f55">Колёсико мыши (нажать)</span> ) --> Сфокусироваться на объекте под курсором
	
* ###  Ходьба ^ad42e9
	* *Шерон автоматически идет к каждой новой метке*
	*  ( <span style="color:#f55">Левая кнопка мыши</span> ) --> Поставить метку
	*  ( <span style="color:#f55">C</span> ) --> Удалить метку и остановить Шерон
	*  ( <span style="color:#f55">Z (зажать/нажать)</span> ) --> Красться
		*  *Режим переключения состояния можно изменить в настройках*
	
* ###  Смена игрового режима
	*  ( <span style="color:#f55">V</span> ) --> Переключиться в режим [[beat-em-up|Beat Em Up]]
	

#  Способности
---

<ul class="tree-list">
	<li><h3>( <span style="color:#f55;display:inline">Правая кнопка мыши</span> ) - Открыть контекстное меню взаимодействия с объектом под курсором</h3></li>
	<ul>
		<li><span style="display:inline-block;margin-left:2em"></span>Осмотреть - Шерон прокомментирует выбранный объект</li>
		<span style="display:inline-block;margin-left:2em"></span><li>Взаимодействовать - виды взаимодействий варьируются:</li>
		<ul>
			<li><span style="display:inline-block;margin-left:4em"></span>Приближенный осмотр - можно крутить объект в 3D/проиграется катсцена</li>
			<li><span style="display:inline-block;margin-left:4em"></span>Головоломка - открывается <a href="obsidian://open?vault=late_night_story&file=visual-style/menus#Интерфейс%20головоломки">интерфейс головоломки</a></li>
			<li><span style="display:inline-block;margin-left:4em"></span>Взлом - камера переключается на <a href="obsidian://open?vault=late_night_story&file=visual-style/menus#Интерфейс%20взлома">мини-игру взлома</a></li>
			<li><span style="display:inline-block;margin-left:4em"></span>Вид - Шерон и все персонажи поблизости сядут спиной к игроку и будут смотреть вдаль, игровой интерфейс плавно затухает до первого ввода пользователя. Камера переходит в красивый ракурс. (толчок сделать снимок экрана)</li>
		</ul>
		<li><span style="display:inline-block;margin-left:2em"></span>Диалог - запускает <a href="obsidian://open?vault=late_night_story&file=visual-style/menus#Интерфейс%20диалога">интерфейс диалога</a>
		Обыскать - спустя время применения, открывает <a href="obsidian://open?vault=late_night_story&file=visual-style/menus#Интерфейс%20хранилища">интерфейс хранилища</a></li>
		<li><span style="display:inline-block;margin-left:2em"></span>Подобрать - перемещает выбранный объект в инвентарь
		Выбросить</li>
		<li><span style="display:inline-block;margin-left:2em"></span>Спрятаться - Шерон прячется в выбранное укрытие
		Пройти - сменить комнату (все персонажи пройдут следом)</li>
	</ul>
</ul>

* ###  Активные способности
	*  ( <span style="color:#f55">Shift</span> ) --> Осмотреться ^f3cb04
		*  *Помечает все несекретные объекты в поле зрения, доступные для взаимодействия*
	*  ( <span style="color:#f55">X</span> ) --> Применить предмет в активном слоте на объекте бод курсором
	


# Механики
---

* ###  Скрытность
	*  *Шерон не атакуют враги пока она не попадет им в поле зрения*
	*  *Шерон хуже видно в темноте*
	*  *Шерон хуже видно в маскировке*
	*  *Шерон легче заметить в режиме [[beat-em-up|Beat Em Up]]*
	*  *Шерон легче заметить при движении*
	*  *Использование громких/ярких предметов привлекает внимание, но не сразу выдает Шерон*
	*  *Нейтральные враги начинают битву, когда понимают что перед ними полицейский*
	*  *[[enemies#^77edde|Агрессивные враги]] атакуют Шерон сразу, как заметят её*
