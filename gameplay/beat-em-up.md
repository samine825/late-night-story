# Управление
---

* ###  Камера
	*  ( <span style="color:#f55">Q</span> ) --> Повернуть поле по часовой
	*  ( <span style="color:#f55">E</span> ) --> Повернуть поле против часовой
	
* ###  Ходьба ^b422c5
	*  *Шерон ходит относительно направлении камеры*
	*  ( <span style="color:#f55">W</span> ) --> движение вверх
	*  ( <span style="color:#f55">A</span> ) --> движение влево
	*  ( <span style="color:#f55">S</span> ) --> движение вниз
	*  ( <span style="color:#f55">D</span> ) --> движение вправо
	
* ###  Прыжок
	*  ( <span style="color:#f55">Пробел</span> ) --> Одиночный прыжок высотой в 2,5 Шерон
	
* ###  Кувырок
	*  ( <span style="color:#f55">Ctrl</span> ) --> Кувырок в направлении взгляда Шерон
	
* ###  Рывок
	*  ( <span style="color:#f55">Shift</span> ) --> Резкий рывок в направлении взгляда Шерон*
	*  *Можно делать в воздухе*
	*  *Тратит энергию*
	


#  Способности
---

* #  Удары на земле
	
	
	* ###  Быстрая атака
		*  ( <span style="color:#f55;display:inline">ЛКМ</span> ) -->  *Комбо битой*
			1.  *Горизонтальный взмах одну сторону*
			2.  *Взмах в другую сторону*
				*  *Каждый удар комбо после второго расходует энергию*
			3.  *Взмах снизу верх*
			4.  *Случайный, ошеломляет и наносит двойной урон*
				*  *Удар битой по макушке*
				*  *Пинок в грудь*
				*  *Удар с выпадом кулаком в нос*
				
			
		
	* ###  Сильная атака ^21f8bd
		*  ( <span style="color:#f55;display:inline">Зажать и отпустить ЛКМ</span> ) --> *Ошеломляющий удар по области впереди*
		*  ( <span style="color:#f55;display:inline">Зажать ЛКМ</span> + <span style="color:#f55;display:inline">F</span> ) --> *Ошеломляющий удар по области сзади*
			*  *Разворачивает Шерон*
		*  *Требует фокусированной зарядки*
		*  *Звуковая и визуальная индикация готовности после зарядки*
		
	* ###  Атака со спины
		*  ( <span style="color:#f55;display:inline">F</span> ) --> *Удар ногой назад*
			*  *Разворачивает Шерон*
			*  *Можно продолжить как обычное наземное комбо со второго удара*
		
	
* #  Удары в воздухе
	
	* ###  Комбо в прыжке
		*  ( <span style="color:#f55;display:inline">В воздухе</span> + <span style="color:#f55;display:inline">ЛКМ</span> ) --> *Комбо в воздухе*
			*  *Расходует энергию с первого удара, четвертый удар всегда слэм*
			*  *Можно перемещаться в воздухе продолжая комбо*
		
	* ###  Слэм
		*  ( <span style="color:#f55;display:inline">В воздухе</span> + <span style="color:#f55;display:inline">Ctrl</span> ) --> *Резкое падение с ударом по области по приземлению*
			*  *Тратит много энергии*
			*  *Подбрасывает врагов снизу*
			*  *Подбрасывает Шерон*
		
	
* #  Специальные
	
	* ###  Апперкот
		*  ( <span style="color:#f55;display:inline">Пробел</span> + <span style="color:#f55;display:inline">ЛКМ</span> ) --> *Подбрасывает врага которого задевает в прыжке*
		
	* ###  Направленное парирование
		*  ( <span style="color:#f55;display:inline">ПКМ</span> ) --> *Парирует синюю атаку в направлении курсора*
			*  *Восстанавливает часть энергии при успешном парировании*
		
	* ###  Блок
		*  ( <span style="color:#f55;display:inline">Зажать ПКМ</span> ) --> *Блокирует удары и снаряды ценой энергии, уменьшая полученный урон*
		
	


# Механики
---

* #  Разновидности вражеских атак
	
	* ### Обычная атака
		*  *Происходит после короткого замаха/прицела*
		
	
	* ###  Сильная атака  ( % ) ^9f5072
		*  *Атака проходит после фокуса*
		*  *Атака пробивает блок*
		*  *При успешном выполнении валит Шерон на землю и сотрясает камеру*
		
	
	* ###  Синяя атака  ( \* ) ^6ca5fd
		*  *Атака подсвечивается синим*
		*  *Шерон может парировать синие атаки, возвращая вред атакующему*
		
	
	* ###  Красная атака  ( ! ) ^a2c027
		*  *При атаке глаза/визоры/атака противника горят красным*
		*  *При успешном выполнении Шерон [[beat-em-up#^21dd36|вырубается]] мгновенно*
		
	
* #  Возрождение ^21dd36
	
	* ###  Проигрыш
		*  *Шерон теряет сознание, камера пикселизируется и затухает*
		
	* ###  Возвращение
		*  *Камера "отматывает" запись до начала драки*
		
	
* #  Ресурсы
	
	* ###  Здоровье
		*  *По истечению здоровья Шерон [[beat-em-up#^21dd36|начинает драку с начала]]*
		*  *Восстанавливается со временем*
		
	* ###  Энергия
		*  *Восстанавливается со временем*
		*  *Восстанавливается дольше, если израсходовать весь запас
		*  *Тратится в гораздо меньшем количестве, если игрок использует приемы в ритм музыки*
		
	
* #  Окружение
	
	* ### Известность
		*  *Чем больше Шерон дерется тем больше становится врагов на локации*
		
	* ###  Предметы окружения
		*  *Шерон может использовать в бою предметы вокруг себя*
