# Механики

* #  Разновидности атак
	
	* ###  Меткая атака  ( " ) ^60b1a3
		*  *Имеет характерный звук*
		*  *Пробивает блок*
		
	
	* ###  Сильная атака  ( % ) ^9f5072
		* *Атака проходит после задержки*
		*  *Валит Шерон на землю*
		
	
	* ###  Синяя атака  ( \* ) ^6ca5fd
		*  *Атака подсвечивается синим*
		*  *Шерон может парировать синие атаки, возвращая вред атакующему*
		
	
	* ###  Красная атака  ( ! ) ^a2c027
		*  *Глаза противника горят красным*
		*  *При успешном выполнении Шерон вырубается мгновенно*
